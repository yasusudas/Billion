Request fields

申請で揃える項目

Purpose

利用目的

要約、分類、下書き、チェックなど、AIを使う業務上の理由を短く書きます。

Scope

対象業務

部署、業務名、作業ステップ、外部影響の有無を分けて記録します。

Data

入力データ

公開情報、社内一般情報、制限すべき情報を分け、投入禁止範囲を明記します。

Review

人間確認

外部送信、公開、請求、契約、権限変更の前で人間確認を置きます。

Rules

禁止事項

根拠不明の断定、機密入力、本人確認情報の投入、例外判断の自動化を止めます。

Evidence

運用記録

実行日、確認者、修正理由、停止判断、次回見直し日を残します。
