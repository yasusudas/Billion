Use order

Suggested sequence

  1. Baseline: Start with AI Ops Diagnostic Kit to define control points and risk ownership.
  2. Growth: Use RFP Radar Setup Kit to create a repeatable opportunity review workflow.
  3. Front office: Use Reception AI Pilot Kit to define inquiry routing, fallback rules, dry-run scenarios, and operator review.
  4. Review: Decide which workflow should expand, pause, or move into a written implementation scope.
